Crypt Warden

Crypt Warden is a password generator built on tkinter using the customtkinter extension.
User will have a few options to choose a character set and from those a random string will be generated,
I recommend using a long password and choose as many character sets as possible to make sure it is strong.
Also in addition to generating a secure string, the user is provided a copy to clipboard button, a dark mode switch.

Crypt Warden will later on migrate to a password manager to be able to store generated passwords securely.
More of this project I will explain in my blog. If you are interested in getting the source code to try Crypt Warden, please visit my Github to get started. It is a pretty straightforward copy-paste process after installing dependencies!
